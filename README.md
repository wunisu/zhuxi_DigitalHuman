# 朱熹 AI 数字人 Demo

这是我基于 **虚幻引擎（Unreal Engine）** 制作的 **朱熹 AI 数字人项目演示**。
项目通过高精度数字人技术与 AI 对话系统结合，实现了一个可以进行自然语言交流、具备实时语音和面部表情反馈的历史人物数字化形象。

该项目旨在探索 **数字人 + AI 交互** 在教育、文化传播与虚拟角色交互中的应用。

## 项目功能

* **AI 对话交互**

  * 通过 AI 对话系统实现实时自然语言交流
  * 支持语音或文本输入并生成对应回答

* **语音生成与播报**

  * AI 自动生成语音内容
  * 数字人进行同步语音播报

* **面部表情与口型同步**

  * 根据语音内容自动驱动面部表情
  * 实现口型与语音的同步动画

* **实时数字人渲染**

  * 使用高精度数字人模型进行实时渲染
  * 支持自然的面部细节与表情变化

---

## 技术架构

### 3D 数字人系统

* **引擎**：Unreal Engine
* **数字人框架**：MetaHuman
* **实时渲染**：UE 实时渲染管线

### AI 对话系统

* **AI 对话平台**：Convai
* **对话逻辑**：基于大语言模型驱动的自然语言交互

### 语音与动画

* **语音生成**：AI Text-to-Speech
* **表情驱动**：语音驱动面部动画 / 口型同步

## 运行方式（本地完整版本）

由于 **完整 Unreal 项目资源与模型文件体积较大（>6GB）**，
本仓库仅提供项目演示与部分示例代码。

如果需要运行完整项目：

1. 安装 Unreal Engine
2. 配置 MetaHuman 插件
3. 配置 Convai AI 对话插件
4. 导入项目资源

完整项目文件可通过联系本人获取。

## 项目亮点

* 使用 **Unreal Engine + MetaHuman** 构建高精度数字人形象
* 集成 **Convai AI 对话系统** 实现自然语言交互
* 实现 **语音生成 + 面部动画同步** 的数字人互动体验
* 将历史人物 **朱熹** 数字化，探索数字人文化传播场景
